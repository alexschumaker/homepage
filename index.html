Hiragana Flashcards

A Javascript/SQLite app I wrote using the Phonegap platform, allowing it to run seemlessly on Android, iOS, Windows, or in a brower.

I designed a specialized shuffling algorithm to make learning phoenetic Japanese more efficient. It will increase the frequency of characters the user often gets wrong while avoiding repetition and still occasionally showing cards that are well practiced.

ActionCamPlus

ActionCamPlus is a UI modification for the popular online game World of Warcraft. It leverages an experimental feature in the game's code to offer a more immersive cinematic experience to players.

I used Lua for the mod's logic and interfacing with the game's API, and XML for the skeleton of the config UI.

Check it out on GitHub or Curse!
